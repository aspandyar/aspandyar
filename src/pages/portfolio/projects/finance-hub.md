## Finance Hub

Finance Hub is a modern, full-stack personal finance management application designed to help users take complete control of their financial life. This project combines a clean, premium user interface with powerful backend functionality to deliver an intuitive and comprehensive financial tracking experience.

Developed with React and TypeScript on the frontend, and Node.js with Express on the backend, Finance Hub focuses on performance, type safety, and a component-centered architecture that makes managing personal finances effortless.

## üß© Features

- **Responsive and minimalist design** - Clean, airy interface with premium appearance
- **Comprehensive Dashboard** - Real-time metrics including balance, income, expenses, savings, and average expenses
- **Transaction Management** - Add, edit, and track income and expenses with categories and custom icons
- **Recurring Transactions** - Automatically handle daily, weekly, monthly, or yearly recurring transactions
- **Budget Predictions** - Create and manage budget predictions for different categories and months
- **Visual Analytics** - Beautiful line charts and donut charts for expenses and income over time
- **Category System** - Customizable categories with icons and colors for income and expenses
- **Transaction History** - Comprehensive history view with filtering and date range selection
- **User Authentication** - Secure JWT-based authentication system
- **Multi-currency Support** - Track finances in different currencies
- **Dark mode aesthetic** - Professional color scheme with green for income, red for expenses

The purpose of Finance Hub is to provide a comprehensive yet intuitive platform for personal finance management.
